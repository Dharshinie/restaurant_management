<b>Culina Suite – Restaurant Management System</b>

Culina Suite is a modern Restaurant Management Web Application designed to streamline restaurant operations such as order management, menu handling, and overall workflow. The system provides an efficient interface for managing restaurant activities with a fast and responsive user experience.

This project is built using a modern full-stack architecture with React, Node.js, and Supabase.

<b>🚀 Tech Stack</b>

<b>Frontend<b/>

->React.js

->Vite

->Tailwind CSS

<b>Backend</b>

->Node.js

->Express.js

<b>Database & Backend Services</b>

->Supabase

<b>✨ Features</b>

->Restaurant dashboard for managing operations

->Menu management system

->Order tracking and management

->Responsive user interface

->Modern and clean UI design

->Fast development using Vite

->Backend API built with Express.js

->Database integration with Supabase

<b>⚙️ Installation</b>

1️⃣ Clone the repository

<b>git clone https://github.com/your-username/culina-suite.git</b>

2️⃣ Navigate to the project folder

<b>cd culina-suite</b>

3️⃣ Install dependencies

<b>npm install</b>

▶️ Run the Project

<b>npm run dev</b>

<b>🗄️ Database Setup</b>

->This project uses Supabase for database and backend services.

->Create a project in Supabase

->Add your database tables

->Connect the project using Supabase API keys

<b>📌 Purpose of the Project</b>

The goal of this project is to build a modern full-stack restaurant management system while learning real-world technologies such as React, Node.js, Express, and Supabase.
